# 遇见信息

遇见信息，打破信息壁垒，让你更容易获得信息。

## 项目介绍

遇见信息是一个基于 Flutter 的社交平台，主要功能有：

- 每日60秒读懂世界
- 新闻简报
- 摸鱼日报
- 摸鱼日历
- 内涵段子
- 星座运势
- 明星八卦
- 历史上的今天
- 抖音热搜
- 头条热搜
- 知乎热榜
- 微博热搜
- Bilbil 热搜
- Bing 每日壁纸
- Epic 免费游戏


## 项目特点

- 基于 Flutter 开发，跨平台，支持 Android、iOS、Web。
- 采用 Material Design 设计风格，简洁、美观。

